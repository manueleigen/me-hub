# Portfolio Website 2025

Kompletter Neuaufbau meiner Portfolio-Website. Fokus auf Performance, minimalistische Ästhetik und flüssige Animationen.

## Ziele

- Case Studies für Aurora, FreeLens und Stadtbibliothek zeigen
- Ladezeit unter 1s (100/100 Lighthouse)
- Vollständig animiert mit Framer Motion
- Kontaktformular mit automatischer Weiterleitung in Notion

## Tech Stack

- Next.js 15 (App Router, SSG)
- Tailwind CSS
- Framer Motion
- Vercel

## Design Entscheidungen

**Farbpalette**: Fast monochrom — Off-White `#F5F4F0`, Ink `#1A1A1A`, Accent `#C8FF00` (Neon-Lime)

**Typografie**: 
- Display: "Cabinet Grotesk" (Variable)
- Body: "Instrument Sans"

**Layout**: Bento-Grid auf der Startseite, Full-Bleed Case Studies
