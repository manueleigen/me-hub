# Figma Plugins — Meine Toolbox

Plugins, die ich täglich oder regelmäßig nutze.

## Unverzichtbar

| Plugin | Zweck |
|--------|-------|
| **Tokens Studio** | Design Token Management, Sync mit Code |
| **Iconify** | Zugriff auf alle Icon-Libraries (Lucide, Heroicons, …) |
| **Stark** | Accessibility-Check (Kontrast, Farbblindheit-Simulation) |
| **Measure** | Abstände und Größen annotieren |
| **Variables Import** | Tokens aus JSON-Dateien importieren |

## Manchmal nützlich

| Plugin | Zweck |
|--------|-------|
| **Unsplash** | Stock-Fotos direkt in Figma |
| **Lorem Ipsum** | Platzhaltentexte generieren |
| **Content Reel** | Realistische Dummy-Daten (Namen, Adressen, Avatare) |
| **Blobs** | Organische Formen für Illustrationen |

## Selbst gebaut / geplant

- **Aurora Sync**: Eigenes Plugin für Aurora → React Token-Sync (in Entwicklung)

## Shortcuts die ich immer vergesse

```
Cmd + G          → Gruppe
Cmd + Alt + G    → Frame aus Auswahl
Cmd + E          → Flatten
Alt + Klick      → Inneres Element auswählen
Cmd + /          → Plugin-Suche
```
