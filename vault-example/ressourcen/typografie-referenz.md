# Typografie-Referenz

Schriften, die ich gerne einsetze — sortiert nach Einsatzgebiet.

## Aktuelle Favoriten

### Display / Headlines
- **Cabinet Grotesk** (Variable) — Geometrisch, modern, läuft aktuell auf Portfolio
- **Clash Display** — Stark, editorial, gut für Landing Pages
- **Syne** — Eigenständig, leicht futuristisch

### Body / Interface
- **Instrument Sans** — Clean, leserlich, perfekt für UI
- **DM Sans** — Neutral, skaliert gut in klein
- **Inter** — Klassiker, nie falsch für SaaS-Produkte

### Mono / Code
- **Geist Mono** (Vercel) — Mein Go-to für Code-Blöcke
- **JetBrains Mono** — Sehr leserlich, guter Kontrast

## Bewährte Pairings

| Display | Body | Gefühl |
|---------|------|--------|
| Cabinet Grotesk | Instrument Sans | Modern, warm |
| Clash Display | DM Sans | Bold, minimalistisch |
| Syne | Inter | Kreativ, professionell |

## Größen-System (Aurora)

| Token | Größe | Line-Height | Verwendung |
|-------|-------|-------------|-----------|
| `text-xs` | 12px | 16px | Labels, Captions |
| `text-sm` | 14px | 20px | Body Small, UI |
| `text-base` | 16px | 24px | Body Default |
| `text-lg` | 18px | 28px | Lead Text |
| `text-2xl` | 24px | 32px | H3 |
| `text-4xl` | 36px | 40px | H2 |
| `text-6xl` | 60px | 64px | H1 / Display |
