# Micro-SaaS Ideen

Sammlung von Produkt-Ideen, die ich irgendwann bauen will (oder nicht).

## 1. Token Studio CLI

**Problem**: Design Tokens aus Figma exportieren und ins Codebase einpflegen ist mühsam.
**Idee**: CLI-Tool, das Figma Variables direkt in CSS Custom Properties, Tailwind Config und TypeScript-Konstanten konvertiert.
**Status**: Proof of Concept läuft lokal
**Aufwand**: ~3 Wochen MVP

## 2. Freelancer Invoice Kit

**Problem**: Rechnungen als Freelancer schreiben kostet mich jedes Mal 20 Minuten.
**Idee**: Einfaches Tool: Kundendaten + Stunden → schickes PDF-Invoice, automatisch per Mail.
**Status**: Idee
**Potenzial**: Breiter Markt, aber viel Konkurrenz (Fastbill, Lexoffice, …)

## 3. Design Review Async Tool

**Problem**: Figma-Kommentare sind chaotisch. Feedback geht verloren.
**Idee**: Strukturiertes Async-Review-Tool für Design-Handoffs — Issues mit Status, Assignee, Severity.
**Status**: Recherche
**Differenzierung**: Fokus auf Designer ↔ Developer Workflow

## 4. Accessibility Checker für Figma

**Problem**: Kontrast und ARIA-Annotations erst im Dev-Handoff prüfen ist zu spät.
**Idee**: Figma Plugin, das WCAG 2.1 AA/AAA direkt in der Design-Phase checkt und annotiert.
**Status**: Idee
**Aufwand**: ~2 Wochen (Figma Plugin API ist gut dokumentiert)
