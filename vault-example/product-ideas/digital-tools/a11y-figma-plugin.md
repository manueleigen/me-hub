# A11y Checker für Figma

Figma Plugin, das Barrierefreiheit direkt in der Designphase sicherstellt — Kontrast, Schriftgrößen, ARIA-Rollen.


## Problem

Accessibility-Prüfungen finden meist erst im Code statt. Fehler werden spät entdeckt und sind aufwändig zu korrigieren.

## Funktionen

- Kontrast-Check nach WCAG 2.1 AA & AAA (Text + UI-Elemente)
- Schriftgrößen-Warnung (< 16px Fließtext)
- ARIA-Role Annotations direkt im Frame
- Export: Accessibility-Report als PDF

## Status

Idea — Figma Plugin API evaluiert, machbar in ~2 Wochen.
