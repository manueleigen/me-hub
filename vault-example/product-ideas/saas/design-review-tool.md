# Design Review Async

Strukturiertes Feedback-Tool für den Designer ↔ Developer Handoff. Kein chaotischer Kommentar-Thread — sondern klare Issues mit Status, Severity und Assignee.

## Problem

Figma-Kommentare gehen verloren, sind unstrukturiert und haben keine Statusverfolgung. Reviews dauern länger als nötig.

## Kernfunktionen

- Issues aus Figma-Frames erstellen (Screenshot + Annotation)
- Status: open / in-review / resolved
- Severity: blocker / major / minor / nitpick
- Assignee + Deadline
- Slack / E-Mail Notifications

## Differenzierung

Fokus auf Designer ↔ Developer Workflow, nicht auf Kunden-Feedback (wie z. B. Pastel oder Ruttl).
